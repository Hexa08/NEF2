## Hardware-Accelerated Vision

Computer vision tasks are heavily dependent on spatial operations like convolutions, pooling, and resizing. NEF2 implements these operations using vendor-optimized libraries (cuDNN for NVIDIA, MIOpen for AMD, MPS for Apple) to ensure peak performance.

## Core Vision Layers

### Convolutions
Highly optimized 1D, 2D, and 3D convolution kernels with support for dilated, grouped, and depthwise operations.

```python
from nef2.vision import nn

# A standard 2D convolution
conv = nn.Conv2d(in_channels=3, out_channels=64, kernel_size=3)
```

### Vision Transformers (ViT)
NEF2 includes first-class support for Vision Transformers. Our implementation features:
-   **Patch Embedding**: Efficiently sharding images into patches via optimized stride kernels.
-   **FlashAttention Integration**: Scaling ViTs to high-resolution images with $O(N)$ memory efficiency.
-   **Learnable Positional Encodings**: Integrated into the `nn.Parameter` system.

## Image Processing Substrate

NEF2 Vision includes a high-speed image processing library for data augmentation and preprocessing. Operations like resizing, rotation, and color space conversion are performed as GPU kernels, avoiding the slow CPU-GPU transfer loop.

```python
from nef2.vision import transforms

# Define a GPU-accelerated pipeline
preprocess = transforms.Compose([
    transforms.Resize((224, 224)),
    transforms.RandomHorizontalFlip(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

# Process a batch of images directly on the GPU
images = preprocess(raw_image_tensors)
```

## Model Presets

NEF2 Vision comes with a library of pre-configured, high-performance model architectures:
-   **ResNet & EfficientNet**: The classics, optimized for low-latency inference.
-   **ViT & Swin Transformer**: For state-of-the-art accuracy in complex scenes.
-   **YOLO-NEF**: A specialized implementation of the YOLO architecture for real-time object detection.

By combining low-level kernel optimization with high-level architectural primitives, NEF2 Vision provides everything you need to build world-class computer vision systems.
