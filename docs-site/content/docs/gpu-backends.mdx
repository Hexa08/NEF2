# GPU Backends

NEF2 provides native support for major hardware accelerators. The framework automatically detects the optimal backend at runtime, or you can force a specific target.

## Supported Architectures

- **CUDA**: Full support for NVIDIA Ampere, Hopper, and Blackwell architectures.
- **ROCm**: Native execution on AMD Instinct MI250/MI300 series.
- **Metal**: Optimized MLX kernels for Apple Silicon (M1/M2/M3).

## Auto-Detection Logic

By default, `nef2.load()` queries the system via the `nef2-sys` daemon. 

```python language-python
import nef2

# Automatically targets 'cuda' if NVIDIA driver is present, 
# falls back to 'rocm', then 'metal', then 'cpu'.
model = nef2.load("model.nef2", device="auto")
```

## Manual Override

You can target specific devices using the PyTorch-like device syntax:

```python language-python
# Target specific GPU
model = nef2.load("model.nef2", device="cuda:1")

# Target Apple Silicon
model = nef2.load("model.nef2", device="metal")
```

<Callout type="info" title="Cross-Backend Compilation">
  The NEFInductor compiler can JIT-compile custom operations for the active backend dynamically.
</Callout>
