## The Module Abstraction

At the core of `nef2.nn` is the `Module` class. It serves as the base class for all neural network components, handling parameter registration, state management, and the `forward` pass logic.

-   **Parameter Tracking**: Modules automatically discover and track `nef2.Parameter` objects within their attributes, making it easy to pass them to optimizers.
-   **Composition**: Modules can contain other modules, allowing you to build hierarchical architectures like Transformers or ResNets through simple composition.
-   **Execution Modes**: Easily toggle between `train()` and `eval()` modes to control behavior like dropout or batch normalization.

## Standard Layers

NEF2 includes a comprehensive set of pre-built layers, all optimized for the NEFCore runtime.

### Linear Layers
```python
from nef2 import nn

# A standard fully-connected layer
fc = nn.Linear(in_features=512, out_features=1024)
```

### Normalization
```python
# Hardware-accelerated LayerNorm
ln = nn.LayerNorm(normalized_shape=512)
```

### Embeddings
```python
# Efficient lookup tables for NLP tasks
emb = nn.Embedding(num_embeddings=50000, embedding_dim=768)
```

## The Pipeline Engine

One of NEF2's unique features is the **Pipeline Engine**, enabled by the `>>` operator. This allows you to chain models together such that they share a **Shared Tensor Bus**, eliminating the need for data serialization between model boundaries.

```python
# Chain models into a seamless pipeline
full_system = vision_model >> reasoning_model >> speech_model

# Execute the entire chain in one call
output = full_system(image_input)
```

## Custom Modules

Creating custom layers is straightforward. Simply inherit from `nn.Module` and define your logic.

```python
class MyBlock(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.ln = nn.LayerNorm(dim)
        self.fc = nn.Linear(dim, dim)

    def forward(self, x):
        return x + self.fc(self.ln(x))
```

By providing a clean, modular API that doesn't sacrifice performance, `nef2.nn` empowers developers to build the next generation of AI models without the friction of traditional frameworks.
