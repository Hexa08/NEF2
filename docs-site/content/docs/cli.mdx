# CLI Reference

The `nef2-cli` tool provides utilities for inspecting, converting, and benchmarking models.

## Commands

| Command | Description |
|---|---|
| `nef2-cli inspect <file>` | Prints the header schema and tensor index of a NEF2 file. |
| `nef2-cli convert <file>` | Converts an ONNX or Safetensors model into the NEF2 format. |
| `nef2-cli bench <file>` | Runs a synthetic inference benchmark and reports token/s latency. |
| `nef2-cli sysinfo` | Displays hardware backend capabilities and topology. |

## Usage Examples

### Inspecting a Model

```bash language-bash
$ nef2-cli inspect llama-7b.nef2

NEF2 Format: v2.0
Tensors: 291
Metadata: 1.2 KB
Alignment: 4096 bytes

[0] model.embed_tokens.weight : f16 [32000, 4096] (262.14 MB)
[1] model.layers.0.self_attn.q_proj.weight : f16 [4096, 4096] (33.55 MB)
...
```

### Converting from Safetensors

<Callout type="tip" title="Optimization">
  During conversion, you can pass `--quantize int8` to automatically compress the model weights.
</Callout>

```bash language-bash
nef2-cli convert model.safetensors --output model.nef2 --quantize int8
```
