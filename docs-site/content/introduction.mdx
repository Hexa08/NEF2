## Core Philosophy

Traditional ML frameworks are built on layers of abstraction that prioritize developer convenience over machine efficiency. NEF2 flips this model:

1.  **Hardware Native**: Direct communication with CUDA, HIP, and Metal via a unified substrate.
2.  **Agent First**: Built-in support for multi-model coordination and shared tensor buses.
3.  **Memory Efficient**: HyperCache technology pages model weights between VRAM, RAM, and NVMe on-demand.
4.  **Language Independent**: Core logic implemented in C++ and Rust for peak performance.

## Key Features

- **Multi-GPU Fabric**: Seamlessly scale models across clusters of mixed hardware.
- **NEFInductor**: A state-of-the-art compiler that performs kernel fusion and graph optimization.
- **RDMA Networking**: Low-latency communication for distributed training and inference.
- **Zero-Copy Serialization**: Move models between disk and memory without expensive conversion steps.

## Quick Start

```python
import nef2
from nef2 import nn

# Define a model in standard Python
class Model(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc = nn.Linear(512, 1024)

    def forward(self, x):
        return self.fc(x)

# Run with hardware-native performance
model = Model()
inputs = nef2.Tensor.randn((1, 512))
outputs = model(inputs)
```
