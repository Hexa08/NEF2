# Compiler Pipeline

**NEFInductor** is the JIT compiler embedded within the NEF2 runtime. It is responsible for converting the lazy execution graph into highly optimized hardware kernels.

## Optimization Passes

When an execution graph is triggered, the compiler performs several passes:

1. **Dead Code Elimination**: Removes operations whose outputs are never consumed.
2. **Kernel Fusion**: Combines multiple pointwise operations (e.g., Add + Relu + Multiply) into a single GPU kernel, drastically reducing memory bandwidth overhead.
3. **Memory Planning**: Pre-allocates VRAM buffers to avoid dynamic allocation during execution.
