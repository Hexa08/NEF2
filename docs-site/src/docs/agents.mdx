## Beyond Single-Model Inference

The future of AI lies in the cooperation of specialized models. NEF2 is the first framework built specifically to support this "Agent-Native" future. We move beyond simple API calls to a deeply integrated substrate where agents can share thoughts, memories, and compute resources.

## The Shared Tensor Bus

The core of NEF2's agent infrastructure is the **Shared Tensor Bus**. This is a high-speed, zero-copy communication channel that allows multiple models to exchange data at the speed of the hardware interconnect.

-   **Zero-Serialization**: Unlike JSON or Protobuf based communication, agents on the Tensor Bus exchange raw tensors. No time is wasted "parsing" or "encoding."
-   **Shared Context**: Multiple agents (e.g., a vision model and a reasoning model) can point to the same KV-cache in VRAM, enabling "shared attention" over the same input.

## Agent Coordination Runtime

The **Agent Coordination Runtime (ACR)** manages the lifecycle and communication of cooperating models. It handles:

1.  **Dynamic Routing**: Routing requests to the most appropriate agent based on current load or expertise.
2.  **Consensus Protocols**: Algorithms that allow multiple agents to "agree" on an output, improving reliability through multi-model verification.
3.  **Resource Arbitration**: Ensuring that competing agents don't starve each other for VRAM or compute cycles.

## Building Multi-Agent Systems

NEF2 provides a clean API for defining and connecting agents into a cohesive system.

```python
from nef2.agent import Agent, Coordinator

# Define specialized agents
vision_agent = Agent(model=ViTModel(), role="perceiver")
reasoning_agent = Agent(model=LLMModel(), role="reasoner")

# Connect them via the Coordinator
system = Coordinator([vision_agent, reasoning_agent])

# The agents now communicate via the Shared Tensor Bus
response = system.solve("Analyze this image and explain the physical dynamics.")
```

## Semantic Memory & HyperCache

Agents in NEF2 utilize **HyperCache** as a long-term semantic memory. Experiences from one agent can be archived into the "Cold" storage tier and retrieved later by other agents, creating a persistent, shared knowledge base for the entire system.

By providing the substrate for collective intelligence, NEF2 is paving the way for autonomous systems that are more capable, efficient, and reliable than any single model could ever be.
